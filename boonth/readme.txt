ocr1.py, ocr3.py, and ocr4.py are the scripts used to solve UserStory1,
UserStory3, and UserStory4, respectively.

lcd.py, numbers.py, and ocr.py contain helper functions.

*_adaptor.py files are used to help testing

test_*.py files perform testing with py.test. For example, to test ocr1.py,

py.test -v test_ocr1.py
